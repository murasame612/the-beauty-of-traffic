#### 2.1 闭塞信号

**2.1.1闭塞的定义**

```
闭塞就是用信号或者凭证，保证列车按照前行列车和追踪列车之间
必须保持一定距离(空间间隔制)运行的技术方法。
指列车进入区间后，使之与外界隔离起来，
区间两端车站都不再向这一区间发车，以防止列车相撞和追尾
```
